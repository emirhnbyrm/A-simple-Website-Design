# Business Coach Landing Page - Bootstrap 5 Re-Code Project
Lead Planner - Business Coach Landing Page (Frontend Study)
Bu proje, profesyonel bir web sitesi tasarımının (Business Coach teması) Bootstrap 5 framework'ü kullanılarak sıfırdan kodlanmış bir çalışmasıdır. Projenin amacı, mevcut bir premium tasarımı referans alarak temiz ve sürdürülebilir bir HTML/CSS yapısı oluşturmaktır.

Proje Amacı
Bu çalışma aşağıdaki teknik becerileri uygulamak amacıyla geliştirilmiştir:

Bootstrap 5 Grid yapısını kullanarak karmaşık mizanpajları oluşturmak.

Responsive (duyarlı) tasarım prensiplerini mobil, tablet ve masaüstü cihazlara uyarlamak.

Bileşen tabanlı (Navbar, Card, Footer) HTML mimarisi inşa etmek.

Bootstrap ikon kütüphanesini ve modern buton tasarımlarını entegre etmek.

Teknik Özellikler
Framework: Bootstrap 5.0.2

İkon Seti: Bootstrap Icons

Tasarım Yaklaşımı: Mobile-First (Mobil öncelikli)

Stil Yapısı: Inline Styles ve Bootstrap yardımcı sınıfları (Utility Classes)

İçerik Bölümleri
Header: Logo ve sağa yaslı navigasyon linkleri ile "Contact Me" butonu.

Hero Section: İki sütunlu yapıda; sol tarafta başlık ve CTA butonları, sağ tarafta görsel alanı.

Services: Gölge efektli (shadow-sm) ve yuvarlatılmış köşeli (rounded-4) kart yapılarından oluşan hizmet tanıtımı.

About Section: Arka plan rengiyle ayrıştırılmış, kurucu biyografisi ve profil görselini içeren detay alanı.

Newsletter Footer: Sosyal medya ikonları, hızlı linkler ve yuvarlatılmış (pill) input alanı içeren bülten abonelik bölümü.

Notlar
Bu proje bir "Clone Coding" çalışmasıdır. Görsel tasarım referans alınmış, tüm kodlama tarafımca sıfırdan yapılmıştır.

Görseller yerel dizindeki /fotograflar/ klasöründen çekilmektedir.

Kurulum
Projeyi incelemek için dosyayı bilgisayarınıza indirdikten sonra index.html dosyasını tarayıcınızda çalıştırmanız yeterlidir.
